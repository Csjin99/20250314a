Flex레이아웃

웹페이지 레이아웃을 효율적으로 구성하기 위한 기술, flexbox를 사용하면
요소들을 부모컨테이너에서 유연하게 정렬하고 배치할 수 있다.
특정요소에 display : flex -> 해당 요소를 flex컨테이너로 사용하겠다.
flex-direction : column,row > 주 축(세로,가로) 방향지정 / 기본은 가로 justify-content : 주 축의 정렬을 어떻게 할것인가? align-items : 교차축을 어떻게 정렬할것인가? (flex-start, flex-end, center, space-around, space-between) flexboxContainer에 flex : number -> 남는 공간에 number에 비율로 할당하겠다.
